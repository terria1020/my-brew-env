## QuickStart

### SET MY MAC WITH BREW

How To setup my MacOS with '[Homebrew](https://brew.sh)'?

1. open Terminal.app
2. run this command:

```bash
/bin/bash -c "$(curl -s https://raw.githubusercontent.com/terria1020/my-dev-env/main/brew_setup.sh)"
```
